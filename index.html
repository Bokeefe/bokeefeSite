 Show of Hands

Show of Hands is a simple browser app that connects all of the users currently on it in real time. Intended for making decisions at large meetings, it has been designed for utter simplicity whether it is being used on a mobile device, tablet, or laptop. Each person gets to cast one "yea or nay" vote and then the vote count and number of voters is updated in realtime up top. When all of the people connected have voted, the stats can be named and stored in the database for reference.

 Current

A MEN stack app made by four developers in a four-day hackathon.

Current was built by Charlotte Taylor, Adam Belliveau, Nate Farnsworth, Brendan O'Keefe. It utilizes Javascript, Node, MongoDB, and HighCharts.js.
The theme of the hackathon was to create an educational tool in the browser. Current's goal was to meter people as they spend the day working on a project and use research on learning to help them improve thier efficiency. Users self-report when the eat, take breaks, have coffee, and Current takes that data in. When there is enough data, the user sees a chart of when they were in their most productive work-flow and the circumstances that lead up to it.
